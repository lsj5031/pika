🔍 PI Performance Diagnostics

This page helps diagnose the frozen UI issue. Click the buttons below to run tests.

📋 Instructions

  1. Keep this page open in a separate tab
  2. Open your PI app in another tab
  3. Wait for the freeze to occur
  4. Switch back to this tab and click "Run Full Diagnostics"
  5. Copy the results and share them

🎯 What to Look For

  • Memory > 500MB: Memory leak or too much data loaded
  • DOM > 3000 elements: Too many messages rendered
  • Long tasks > 100ms: JavaScript blocking the main thread
  • FPS < 30: UI can't keep up with updates
